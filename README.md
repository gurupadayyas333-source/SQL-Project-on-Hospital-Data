# SQL Hospital Data Analysis Project

## Overview

This project focuses on analyzing hospital operational and financial data using PostgreSQL. The database contains information about hospitals, departments, patient counts, doctor availability, admission and discharge dates, and medical expenses. Through a series of SQL queries, the project extracts valuable insights that can help healthcare organizations improve resource allocation, patient care, and financial planning.

## Project Objectives

* Design and manage a hospital database using SQL.
* Analyze patient volumes across hospitals and departments.
* Evaluate doctor availability and hospital performance.
* Monitor medical expenses and healthcare costs.
* Generate meaningful reports for operational decision-making.

## Database Structure

The project utilizes a hospital dataset containing:

* Hospital Name
* Location (City)
* Department
* Doctors Count
* Patients Count
* Admission Date
* Discharge Date
* Medical Expenses

## SQL Concepts Applied

* Table Creation and Data Management (DDL)
* Aggregate Functions (SUM, AVG, COUNT)
* Data Filtering and Sorting
* GROUP BY and ORDER BY Clauses
* Date Calculations and Analysis
* Ranking and Top-N Queries
* Healthcare Performance Reporting

## Key Analyses Performed

* Total number of patients across all hospitals
* Average number of doctors available per hospital
* Top 3 departments with the highest patient count
* Hospital with the highest medical expenses
* Daily average medical expenses by hospital
* Longest hospital stay analysis
* Total patients treated per city
* Average patient stay duration by department
* Department with the lowest patient count
* Monthly medical expense trends and reporting

## Skills Demonstrated

* PostgreSQL
* SQL Query Writing
* Data Analysis
* Healthcare Data Analytics
* Business Reporting
* Date and Time Functions
* Data Aggregation and Insights Generation

## Project Outcome

This project demonstrates how SQL can be used to transform raw healthcare data into actionable insights. By analyzing patient trends, department performance, hospital expenses, and treatment durations, the project showcases practical applications of SQL in healthcare analytics and operational reporting. It serves as a strong portfolio project for aspiring Data Analysts, Business Analysts, and SQL Developers interested in real-world data analysis scenarios.
